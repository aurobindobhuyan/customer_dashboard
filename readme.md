# Customer Dashboard


### Tech Stack

- 💻 JavaScript | ES6
- 🌐 ReactJS
- 🔧 Git | Markdown
- 📦 lodash, react, react-bootstrap, react-google-charts
### Features

 - And with that raw content we are displaying all the users and their order details.
